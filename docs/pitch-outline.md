### Слайд 2: Проблема
**Текст:**
- Технические собеседования занимают много времени рекрутеров
- Субъективность оценки кандидатов
- Невозможность масштабировать процесс интервью
- Отсутствие адаптивности под уровень кандидата
- Проблемы с читингом на удалённых интервью

**Визуал:**
- Статистика: среднее время на интервью, количество кандидатов
- График роста потребности в разработчиках

---

### Слайд 3: Решение
**Текст:**
- Автоматизированная платформа с ИИ-интервьюером
- Адаптивная генерация задач под уровень кандидата
- Браузерная IDE с изолированным выполнением кода
- Автоматический анализ кода и генерация отчётов
- Система анти-читинга

**Визуал:**
- Скриншот платформы
- Иконки ключевых фич

---

### Слайд 4: Архитектура
**Текст:**
- **Frontend**: React + TypeScript, Monaco Editor
- **Backend**: FastAPI (Python)
- **Runner**: Docker для изоляции выполнения кода
- **LLM**: Scibox для генерации задач и анализа
- **Database**: PostgreSQL для хранения данных

**Визуал:**
- Диаграмма архитектуры (из docs/architecture-diagram.mmd)
- Стрелки между компонентами

### Слайд 5: Демо-фичи
**Текст:**
1. **Генерация задач**: ИИ создаёт задачи под уровень (Junior/Middle)
2. **Браузерная IDE**: Полнофункциональный редактор кода в браузере
3. **Выполнение кода**: Изолированное выполнение в Docker с ограничениями
4. **Чат с ИИ**: Адаптивное общение, уточняющие вопросы
5. **Автотесты**: Видимые и скрытые тесты для объективной оценки
6. **Финальный отчёт**: Структурированные метрики + текстовый отзыв

**Визуал:**
- Скриншоты каждого фича
- Или короткое видео демо
